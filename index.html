My name is Ahsan Azim!

I'm from Lahore (Pakistan), although my family's moved around quite a bit - I've lived variously in New York, the Netherlands, and Italy.

I'm currently an undergrad at Dartmouth College in Hanover, New Hampshire. I'm studying Computer Science, Math, and Government, and plan for a career in tech!

More specifically, I'm interested in Web development, cross-platform app development, data analysis, as well as pure mathematics and comparative politics.

This website itself is rather curious - I decided to eschew using any of the in-vogue third-party libraries, and stuck to good old jquery for animating things. Indeed, despite personally swearing by the excellent React.js library, jquery was fun and fast to use for something of this scale, and I enjoyed the process.

If you're curious, the code is housed here on github.
